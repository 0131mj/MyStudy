# Basic Elements

ABC 언어의 구성요소는 3가지다.



### statements 문

컴파일러가 해석 후, 명령으로 바꾸어주는 대상

- 공문, 식문, 제어문, 선언문
- 단문, 중문



### Expressions 식

컴파일러가 해석 후, 값으로 바꾸어주는 대상

- 값식, 연산식, 호출식



### Identifier 식별자

식의 위치에 대한 식별자

- 기본형, 참조형
- 변수, 상수



```javascript
const a = 3;
// const : 문(명령): "값을 할당해라"
// a : 식별자
// 3 : 식
```



#### - 변수

##### 역할에 따른 변수의 종류

```javascript
const v = 100;
let acc = 0;
for(let i = 100; i > 1; i--){
    acc += i;
}
acc +=1;
```

위 코드의 역할에서 변수는 다음과 같다. 

- v : 상수, 함수차원에서, 주어진 인자는 상수와 같다. 즉, 주어져 해결해야하는 과제에서 미리 주어진 값이다.
- acc: 스토리지(저장소)
- i : 카운터 (제어변수의 일종)
