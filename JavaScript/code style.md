# Code Style

1부터 100까지의 숫자의 합을 더하는 문제는 이렇게 만들 수 있다. 



### 1부터 100까지 더하기

```javascript
const sum100 = () => {
    let acc = 0;
    for (let i = 1; i <= 100; i++) {
        acc += i;
    }
    return acc;
}
```



### 1부터 더해가다가 100이 되면 멈추기

문제의 재해석

1부터 100까지 거꾸로 더하기

1부터 100까지 차례차례 더하기

어디서부터 어디까지 더할지를 받아서, 실행하도록 만들고, 범위를 받으면 실행하기
