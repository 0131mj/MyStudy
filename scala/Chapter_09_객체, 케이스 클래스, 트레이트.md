## 1. 객체

- 인스턴스를 단 하나만 가지도록, 강제하는 형태의 클래스 (객체지향에서는 싱글톤 singleton 이라고 함.)
- new 키워드로 인스턴스 생성 ( X ), 이름으로 직접 해당객체에 접근 ( O )
- 실행중인 JVM에서 최초로 접근할 때 자동으로 인스턴스화 됨. (객체에 접근하기 전까지는 인스턴스화 되지 않음.)



##### 구문 : 객체 정의 

```scala
object <식별자> [extends <식별자>] [{ 필드, 메소드, 클래스 }]
```
